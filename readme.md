## DEPLOYMENT

Git clone this application and run  locally 

```bash
  git clone git@github.com:SUVAJITKARMAKAR/TUNES-MUSIC-LOADOUT.git
```

```bash
  pip install flask
```

To run the application 
```bash
  python app.py
```
